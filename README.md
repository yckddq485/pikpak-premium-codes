## Invitation Code vs. Redeem Code: What's the Difference?

Here's where people get confused. There are two completely different types of codes in the PikPak ecosystem, and they work very differently.

**Invitation Code (also called Referral Code)**

This is what you enter *when you first sign up*. An invitation code is tied to an existing user's referral link. When a new user registers using someone's invitation code, there's a chance they receive a short free Premium trial — typically around 5 days. The key word is "chance" — PikPak's official documentation states that free Premium through invitation codes is not guaranteed for all users; it depends on whether the inviter has active Special Reward Quota.

Invitation codes are free to use. You enter one during registration, and either you get a bonus or you don't — either way, your account is created normally.

**Redeem Code (also called Voucher Code)**

A redeem code is a purchased code that grants a specific duration of Premium membership when activated. Think of it like a gift card for PikPak Premium — you buy a code (say, for 30 days or 365 days of Regional Premium), and when you enter it in the redemption portal, that duration gets added to your account.

Redeem codes are purchased from authorized resellers, not handed out freely. PikPak officially lists two authorized reseller channels: Weidian and AP Seller.

## A Few Real Caveats

PikPak has a solid product, but a few things are worth knowing going in.

**The September 2024 outage.** A data center fire knocked the service offline for over nine days. PikPak compensated users (9 free days for free users, 18 days for Premium users), which was a reasonable response — but if you need absolutely critical 24/7 availability for work-critical files, keep a backup plan.

**iOS bulk download limitation.** The iOS app doesn't support downloading multiple files simultaneously. You're stuck doing one file at a time, which gets old fast with large libraries. Use the web app or desktop client for bulk operations.

**Region detection.** PikPak determines your tier by combining IP address, device language, and SIM card country code. If you use a VPN regularly or have an account bound to a phone number from a different country, the detection can misfire. Fix it by binding a phone number from your actual current country.
